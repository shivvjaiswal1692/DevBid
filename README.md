# DevBid

DevBid is a cross-platform mobile marketplace application developed using Flutter, designed to help developers and students monetize their digital products and projects.
<br>
The platform enables users to list apps, SaaS products, websites, AI tools, and software projects, while buyers can explore listings, place bids, negotiate deals, and securely acquire products.
DevBid includes features such as authentication, real-time chat, bidding system, secure payment integration, and structured project delivery workflow. The application focuses on solving the problem of unused or non-monetized developer projects by creating a trusted ecosystem for digital asset exchange. The backend architecture is built using Node.js and MongoDB, with Firebase for authentication and notifications. The project emphasizes scalable system design, clean UI/UX, secure transactions, and practical marketplace workflows, making it a real-world startup-oriented product rather than a traditional academic project.
